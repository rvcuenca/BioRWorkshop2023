Installing Packages

We now install the necessary package we need for this workshop called tidyverse. There are two ways we can do this:

  1. Go to the console pane, type the following code and hit Enter.

    install.packages("tidyverse")

  2. Alternatively, one go to Tools > Install Packages and type on Packages blank entry the name of the package you want to install, e.g. tidyverse. Once done click “Install”.

Loading Packages

There are two ways to load a package in a current session of R.

  1. Using the Console, type the following command line and hit Enter.

    library(tidyverse)

  2. Using Packages tab in the Output Pane, first search the name of the package (assuming it is already installed) in the mini-search bar. Check the corresponding check box that matches to the package you want to load in the current session of R.

When you installed R for the first time in you system, some packages are already pre-installed along with it. Some of these packages are even pre-loaded when you open RStudio.
